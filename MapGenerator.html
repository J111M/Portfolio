Map Generator

A random terrain generator made by following a Sebastian Lague tutorial. I am really interested in procedural generation but it is very complicated so I searched around on the internet for some help and came across his educational tutorial series which helped me to understand how the basics of procedural generation work.

How does it work

We first generate a random perlin noise so we can give the terrain interesting shapes. The next step is to make a colour map of the noise so we can check how light or dark a certain place is by giving it a value of 0.0 to 1.0, 1.0 is the lightest point and 0.0 is the darkest point of the noise. We can give certain points between these values colour. The same method is used for the height of the mountains the lighter the noise the higher the mountain and the darker the noise the lower.

What did I learn from this project

I learned how a random terrain is generated behind the scenes and how complicated it is. I learned different methods to generate a terrain like perlin noise or cellular automata. And I also learned a lot of small features in unity that i didn't even know existed. So the projects goal of learning more about map generation is fulfilled. The project itself is still not done I want to make the terrain infinite but that is gonna cost me a lot of time. I still have a lot of fun working on it and there is so much more to learn.
