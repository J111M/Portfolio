Active Ragdoll

I made a player controller with a ragdoll. This makes it able to interact with the physical environment. This way of movement is made with 2 bodies. The first body plays the animation and can not be seen in the scene this body is under the ground and it's only function is to play the animations. The second body is a ragdoll, all the bones of this body are connected to each other. This is the body uses the animations to make a physics based walk.

Configurable Joints

All the bones of my 3D character are connected via configurable joints. Otherwise the arms and legs will just fall to the ground like water. Almost all the main bones have a collider on them so when it gets touched by an object it is realistic.

Walk Animation

This script copies the limbs from the animated body to the physical body. So when an animation gets played the ragdoll(physical body) wil move like the animated body. This script doesn't make the character stand up right this is for animation only.
