Island Survival Game

This is a project that I'm working on with a friend who does the art and I do all the coding. The point of the game is to survive on an island. You have to gather ores, food, wood and kill monsters to survive. We are still working on a crafting system, a building system and something to make your island bigger but that will come in the future. We work on this project mainly when we have spare time so the development is not really fast but still fun nonetheless.

Inventory system

A hard part of the game to code was the inventory system because I have never looked into this before you have to store items and make them able to switch places. It also needs to be able to pick up and drop items when the player is close enough. There are a lot of different steps that make it hard but I managed to get the base of the inventory down.

What did I Learn from this project

What I learned from this project is how the basics of object oriented programming works in C#. I used it for the different tools in the game because a lot of the logic is the same the main difference is what they can mine, chop or slash. I also learned how to communicate better with an artist. At first it was a little hard to get the assets in the right formats so they work in unity but after a while things went a lot smoother and I'm sure this experience will help me in future projects.
